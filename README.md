## API Reference

#### Get all items

```http
  https://api.covid19api.com/summary
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `https://api.covid19api.com/summary` | `data.Global.NewConfirmed` | **Required**. Your API key |

#### Get item

```http
  GET /api/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
